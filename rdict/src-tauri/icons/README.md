# Icons

Place your application icons here:

- `icon.png` - Main icon (512x512 recommended)
- `icon.ico` - Windows icon
- `icon.icns` - macOS icon
- `32x32.png` - 32x32 PNG icon
- `128x128.png` - 128x128 PNG icon
- `128x128@2x.png` - 256x256 PNG icon for retina displays

You can generate these icons from a single high-resolution image using tools like:
- https://github.com/tauri-apps/tauri-icon
- https://iconverticons.com/
