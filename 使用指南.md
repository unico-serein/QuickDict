# EzDict2 使用指南

## 应用已启动 ✓

应用现在已经在后台运行。你会看到两个窗口：
1. **配置窗口** - 显示词典状态和测试查询功能
2. **查询窗口** - 当查询单词时会自动弹出

## 快速开始

### 方式一：使用快捷键查询
1. 复制任何英文单词（例如：复制 "hello"）
2. 按下 `Ctrl + Alt + D`
3. 查询窗口会自动弹出并显示释义

### 方式二：自动查询
1. 直接复制英文单词到剪贴板
2. 应用会自动识别并查询
3. 查询窗口会自动弹出

### 方式三：测试查询
1. 在配置窗口的输入框中输入单词
2. 点击 "Lookup" 按钮
3. 查看结果

## 常见问题

### Q: 首次查询很慢？
A: 首次加载词典需要几秒钟，之后的查询会很快。

### Q: 查询不到单词？
A: 确认单词拼写正确，或者词典中确实包含该单词。

### Q: 图片或音频无法显示？
A: MDD资源文件可能正在加载中，请稍等片刻。

### Q: 如何更换词典？
A: 修改 `src/main.js` 中的词典路径配置。

## 技术细节

- **框架**: Electron + Node.js
- **词典解析**: js-mdict
- **UI渲染**: HTML + CSS
- **通信**: IPC (进程间通信)
